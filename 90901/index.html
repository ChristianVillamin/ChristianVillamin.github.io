Command Line

echo

$ echo {text}

Prints the argument

pwd (Print Working Directory)

$ pwd

Prints current directory

cd (Change Directory)

$ cd {absolutePath}

Go to the desired path

$ cd ./{relativePath}

. is current directory

$ cd ../{relativePath}

.. is above directory

$ cd ~/{relativePath}

~ is home directory

$ cd -/{relativePath}

- is previous directory

ls (List Directories)

$ ls

Lists files and directories of current path

$ ls {path}

List files and directories of target path

$ ls -a {path}

List files and directories including hidden

$ ls -l {path}

List files and directories in detail

$ ls -la {path}

Combination of "ls -a" and "ls -l"

touch

$ touch {fileName}

Creates a new empty file

file

$ file {fileName}

Check file type and show content

cat (Concatenate)

$ cat {fileNameA} {fileNameB}

Shows contents of selected files

less

$ less {directory}

Displays content in a paged manner

history

$ history

Shows history of commands

clear

$ clear

Clears the terminal

cp (Copy)

$ cp {fileName} {directory}

Copies the file to the directory

$ cp *.{extension} {directory}

Copies all file with same extension

mv (Move)

$ mv {file} {directory}

Moves file to the directory

$ mv {oldFile} {newFile}

Renames the file

mkdir (Make Directory)

$ mkdir {folder1} {folder2} {...}

Creates directories/folders

rm (Remove)

$ rm {fileName}

Removes a file

$ rmdir {directoryName}

Removes a directory

find

$ find {path} {type} {fileName}

Finds a file in a directory

help

$ help {command}

Checks what the command is for and its flags

man

$ man {command}

Checks what the command is for and its flags

whatis

$ whatis {command}

Display one-line manual page descriptions

alias

$ alias {name}='{command}'

Create an alias for commands

$ unalias {name}

Removes the alias

exit

$ exit

Exit from the shell

Text-Fu

stdout (Standard Out)

$ echo {text} > {file}

'>' redirects text to file

$ echo {text} >> {file}

'>>' prevents overwritting

stdin (Standard In)

$ cat < {fileA} > {fileB}

'<' as stdin gets '>' to target

stderr (Standard Error)

$ ls /fake/dir > {file}

Invalid, no directory

$ ls /fake/dir 2> {file}

'2>' allows writes err message in file

pipe and tee

$ ls -la /etc

This is long and hard to read

$ ls -la /etc | less

Pipe operator to attaches command

$ ls -la /etc | tee {file}

'tee' allows different output streams

env (Environment)

$ env

Prints environment variables

cut

$ cut -c {n} {file}

Prints nth character in each lines

paste

$ paste -s {file}

Prints all content in one line

head

$ head {file}

Prints the first 10 lines of a file

tail

$ tail {file}

Prints the last 10 lines of a file

expand and unexpand

$ expand {file}

Converts tabs to spaces

$ unexpand -a {file}

Converts spaces to tabs

join and split

$ join {fileA} {fileB}

Combines multiple files together

$ splt {file}

Splits to new file every 1000th line

sort

$ sort {file}

Sort lines by a-z order

$ sort -r {file}

Sorts in reverse order

$ sort -n {file}

Sorts by numerical value

translate

$ tr [a-z] [A-Z] < {file}

Capitalizes all characters

uniq (Unique)

$ uniq {file}

Removes duplicate values

$ uniq -u {file}

Gets unique values

$ uniq -d {file}

Gets duplicate values

wc (Word Count)

$ wc {file}

Checks the total count of words in a file

nl (Number Lines)

$ nl {file}

Checks the total count of lines in a file

grep

$ grep {value} {file}

Check if value exists in file

$ grep -i {value} {file}

Check again but case insensitive

User Management

sudo

$ cat /etc/shadow

Notice this denies permission

$ sudo cat /etc/shadow

'sudo' gives root access and allows

su

$ su {command}

Run 'sudo' via other identity. Not recommended

/etc/passwd

$ cat /etc/passwd

List detailed information of users

/etc/shadow

$ sudo cat /etc/shadow

User auth storage. Requires superuser.

/etc/group

$ cat /etc/group

Allows different permission for different groups

Adding Users

$ sudo useradd {username}

Adds a new user into the system

Removing Users

$ sudo userdel {username}

Deletes the user from the system

Changing Password

$ sudo passwd {username}

Changes a user's password

Permissions

chmod (Modifying Permissions)

$ chmod u+x {file}

Adds executable permission for user

$ chmod u-x {file}

Removes executable permission for user

chmod (Multiple Permissions)

$ chmod ug+w {file}

Adds multiple permission for user

chmod (Numerical)

$ chmod 755 {file}

Grants all permission for user

Modify User Ownership

$ sudo chown {user} {file}

Sets owner of file to user

Modify Group Ownership

$ sudo chgrp {group} {file}

Sets the group of file to selected

Modify user and group simultaneously

$ sudo chown {user}:{group} {file}

Does both modifier for the file

Umask

$ umask 021

New files has all permissions granted

Setuid

$ passwd

Change user password

Setgid

$ sudo chmod g+s {file}

Allows to run as if a member of group.

Processes

ps (Processes)

$ ps

Shows the current processes

$ ps l

Current processes in long format

$ ps aux

Current processes in detal

top

$ top

Shows real time information

kill (Terminate)

$ kill {processID}

Sets owner of file to user

Niceness

$ nice -n {val} apt upgrade

Sets priority value for new processes

Renicing

$ renice {val} -p {processID}

Set priority for existing processes

/proc filesystem

$ cat /proc/{processID}/status

Check process state information

Job Control

$ sleep {processID} &

Allow process to run in background

Background Jobs

$ jobs

Views all running jobs

Background to Foreground

$ fg

Switch to FG most recent job

$ fg %{jobId}

Switch to FG selected job

Kill Background Jobs

$ kill %1

Terminate selected job

$ kill $(jobs -p)

Terminate all jobs

Packages

gzip

$ gzip {file}

Compresses a file down

gunzip

$ gunzip {file}.gz

Decompresses a .gz file

Archiving with tar

$ tar cvf {myTar}.tar {file1} {file2}

Archive through tar

Unpacking archives with tar

$ tar xvf {myTar}.tar

Extracts a tar acrhive

Compressing archive with tar

$ tar czf {myTar}.tar.gz

Creates a compresses tar file

Uncompress and unpack tar

$ tar xzf {myTar}.tar

e'X'tract all 'Z'ee 'F'iles :)

dpkg (Debian Package)

$ dpkg -i {package}.deb

Installs a debian package

$ dpkg -r {package}.deb

Removes a debian package

apt (Advanced Packaging Tool)

$ apt install {package}

Installs package from a repository

$ apt remove {package}

Removes package from a repository

List Debian Packages

$ dpkg -l

List installed debian packages

Update Packages

$ apt update; apt upgrade

Updates repository packages

Packages Information

$ apt show {packageName}

Check installed package information
